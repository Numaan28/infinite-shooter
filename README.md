# infinite-shooter
Infinite Shooter is a fast-paced 2D survival shooter where reflexes and strategy decide how long you last. Set in a minimal, abandoned arena, players must eliminate incoming enemies within a limited time while managing health and positioning. The game features multiple difficulty modes, a fully customizable mission option, smooth continuous firing, and intelligent enemy pursuit. Designed for both PC and mobile, it offers keyboard-and-mouse controls on desktop and intuitive touch controls on mobile devices. With responsive gameplay, clean visuals, and increasing pressure every second, Infinite Shooter delivers an intense arcade experience that challenges accuracy, speed, and focus.
